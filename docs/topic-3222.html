Common Skills que tout les rôles ont:

Break fall: Permet de limiter vos dommages lors de chute
Disabling blow: Attaque de melé qui désactive le parry de votre adversaire
Disabling shot: Attaque d'archery qui désactive le parry de votre adversaire
Gank: Permet d'achever votre adversaire
Heal mount: Heal pour monture
Heal self: Heal pour soi
Light: Lumière
Repair: Réparer des structures (Murs, Bateaux etc)
Rest: Méditation pour reprendre vos points de vie, mana ou stamina
Revive: Revive un allié qui est en down state
Health to mana: Conversion
Mana to stam: Conversion
Stam to health: Conversion
Unburden: Permet le déplacement lorsque vous êtes encombré
