Example de macros pour G15 / Autohotkey / Nostromo

J'ai ajouté en pièces jointes mon fichier de Autohockey que j'utilisais dans le passé.

Voici le code de base pour: "Quick Loot, Équiper sword et shield sur une seul touche et un cycle de trois spells sur la même touche".

La Section du haut c'est la déclaration des constantes, variables et timer etc.
Code:
;Darkfall Scripts ; Defined keys: ; Mouse Button 4 = Switch to sword/shield ; Ctrl-V = Put weapon away and rest ; Shift x 2 = Auto Sprint - Shift again to turn it off ; Alt-RClick = Set quick loot location ; Alt-LClick = Move item under the cursor to set location #NoEnv #SingleInstance force SetWorkingDir %A_ScriptDir% SendMode Input OnExit, DoExit currentButtonCycle := 0 ;start button cycle on hotkey 7 currentButtonCycle_1 := 2 currentButtonCycle_2 := 3 currentButtonCycle_3 := 0 currentButtonCycle_4 := 5 currentButtonCycle_5 := 0 currentButtonCycle_6 := 5 currentButtonCycle_7 := 6 ;Timers cycleResetTimer := 15000 cycleResetTimer1 := 15000 cycleResetTimer2 := 10000 cycleResetTimer3 := 10000 cycleResetTimer4 := 15000 cycleResetTimer5 := 15000 cycleResetTimer6 := 15000 cycleResetTimer7 := 15000 ;CONSTANTS PLAYSOUNDS := True ; (true/false) plays a short beep to signal the macro hotkey has completed iniFile := A_ScriptName . ".ini" ; settings file used for quick loot shortDelay := 250 ; (milisec) delay used between switching hotbars and selecting hotkeys. Seems like the delay doesn't need to be very long commandDelay := 600 ; (milisec) delay between using a command. ie. Switching weapons. This delay needs to be longer and may need to be adjusted for lag ;Uses an ini file setting to determine if the script is already running when launched a second time ; allowing the script to toggle on/off from one shortcut. Created to tie a single G15 key to launch / close the script IniRead, scriptAlreadyRunning, %iniFile%, Settings, scriptAlreadyRunning, False If scriptAlreadyRunning = True { SoundPlay, closing.wav, Wait ExitApp } SoundPlay, started.wav IniWrite, True, %iniFile%, Settings, scriptAlreadyRunning ; Quick Loot Setup IniRead, xdest, %iniFile%, Loot, 1300 ; x destination used for quick looter IniRead, ydest, %iniFile%, Loot, 700 ; y destination #IfWinActive, Darkfall Online ; make sure all the hotkeys only work inside the game ;**************************************** ; Quick loot ; Alt+Right Click - sets the location ; Alt+Left Click - moves item under cursor to set location ; $!Lbutton:: MouseGetPos, x1, y1 MouseClickDrag, left, x1, y1, xdest, ydest, 0 ;0 is mouse speed, 0 is instant MouseMove, x1, y1, 0 ;0 is mouse speed, 0 is instant Return ;Loot destination setup $!Rbutton:: MouseGetPos, xdest, ydest IniWrite, %xdest%, %iniFile%, Loot, xdest IniWrite, %ydest%, %iniFile%, Loot, ydest Return playBeep: If (PLAYSOUNDS) SoundBeep, 900, 50 Return ;Called when the script is exited. Resets the ini setting used to determine if the script is being run ; a second time to enable a toggling ability. It skips the reset if the script is being reloaded through #SingleInstance force DoExit: If A_ExitReason != Single IniWrite, False, %iniFile%, Settings, scriptAlreadyRunning ExitApp ;**************************************** ;Sword/Shield in slots 1 and 2 *~X:: Suspend, On ; suspends hotkeys so only one can run at a time so they don't screw each other up Send, {RCtrl down}0{RCtrl up} ; sends hotkey associated with the sword Sleep, %commandDelay% ; wait a short time for sword to equip Send, {RCtrl down}1{RCtrl up} ; sends hotkey associated with the shield Gosub, playBeep Suspend, Off ; turns all hotkeys back on Return ;**************************************** ;Cycle stam / mana / leech ;Cycle de plusieurs spells sur la même touche. *~E:: Send, {LAlt down}%currentButtonCycle%{LAlt up} Sleep, 100 currentButtonCycle++ if (currentButtonCycle > 2) { currentButtonCycle := 0 } if (cycleResetTimer > 0) { SetTimer, ResetCycle, -%cycleResetTimer% } Return ResetCycle: currentButtonCycle := 0 Return

Si vous aimez le jeux, je peux vous coder des macros pour les touches que vous voulez.

Bouton F1 = Cycle de 2 à 5 spells
Etc.
