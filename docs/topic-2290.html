Tutoriel pour streamer xSplit/Twitch.tv

Message #1 le 2012-11-29 @ 22:38:30 par Xarek:
Ce tutoriel s'adresse aux leaders de division qui voudraient streamer un match.

Étape 1 : téléchargez xSplit

Étape 2 : Lancez l'installateur et assurez-vous de cocher tous les logiciels qu'il faut.

Étape 3 : Ouvrir xSplit avec les informations suivantes :

- Username : admin@armata.ca
- Password : tenebrae

Étape 4 : Ajoutez le canal ArmataTV en cliquant sur "Broadcast" dans le menu du haut, puis sur "Edit Channels".

- Cliquez sur le bouton "Add", puis "Justin.tv/Twitch" dans la fenêtre qui vient d'apparaître.

Étape 5 : Entrez les informations du compte Twitch.tv :

- Username : armatatv
- Password : warhammer
- Ajustez les autres options à votre guise (essai-erreur).
- Appuyez sur "OK"

Étape 6 : Ajoutez du contenu en cliquant sur "File" dans le menu du haut. Tout dépendant du type de contenu streamé, le logiciel va vous guider dans la procédure.

Étape 7 : Une fois que votre scène est complète, cliquez sur "Broadcast" dans le menu du haut, puis sur "armatatv".

- Vérifiez que tout fonctionne en accédant à cette page : http://www.twitch.tv/armatatv
- Pour STOPPER le streaming, refaite l'étape 7.

ATTENTION : par défaut, le logiciel stream le son de votre ordinateur. Pour couper le son, cliquez sur les icônes de microphone et de son dans le bas de xSplit.

Voila. Le logiciel est très complet et je ne peux couvrir avec vous toutes les fonctionnalités. Si jamais vous avez une question, n'hésitez pas à m'envoyer un PM sur Teamspeak.

Bon streaming !
