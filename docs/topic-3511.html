Composition du raid de progression Armata

Après plusieurs raid fait par Armata dans les dernières semaines, il est maintenant temps de désigner les raiders officiels fesant parti du groupe de progression.

Voici quelques pointsque nous recherchons :

1- Disponibles les Jeudi-dimanche- Lundi
2- Vouloir maximiser ses performances (gear-gameplay-soul etc..)
3- respecter les règlements de raid
etc.

Alors, si vous avez les qualitées et le vouloir de tout downer dans le jeu, dans un groupe organisé et optimisé, répondez a ce post en écrivant le numéro de la place dans le raid disponible. Si jamais il y a une journée que vous n'êtes pas disponiles vous pouvez l'inscrire mais une priorité aux personnes disponible les 3 journée sera tenu en compte.
Si vous aimez plus être remplaçant, inscrivez-le.

C'est a noter que vous devez essayer de maximiser vos performances dans tous les rôles que vous serai attitrés, ceci a pour but de former un groupe versatile qui peut s'ajuster a toute situations.

1- Tank warrior/dps
2- Tank warrior/dps
3- dps/ rogue tank
4-Archon/dps (ou chloro)
5-Chloro/dps
6-Chloro/dps
7-Bard/dps
8-dps/Bard
9- Cleric healer/dps
10- Cleric healer/dps
11- Cleric healer/dps
12- Dps /cleric Senticar
13- dps/tank
14- dps mage / Archon
15- dps range / + autre rôle
16- dps range / + autre rôle
17- dps range / + autre rôle
18- dps melee / + autre rôle
19-dps melee / + autre rôle
20-dps-melee / + autre rôle

(les places 12 a 20, la majorité du temps, vous serez assignés dps, mais ayez un autre rôle le plus optimisé possible en cas d'absence ou de stratégie de fight)
