Message #7 le 2013-8-5 @ 19:26:49 par Hecth:
New patch

New Additions

Zombie Ai
Added Flanking Zombies
Added Sprinting Zombies
Detailed information here: http://forums.infest...iary-zombie-ai/
Added additional Zombie spawns to Smallville, Clearview, and Campos.
Added a "playtime" limit to Private servers. You can now set the amount of time a player has played to get into your server. IE If you set the playtime to 1000 a player will need 1000 hours in order to join your server.
An icon will be next to the map name that signifies it has a gametime limit.
There is now a filter for gametime in the server menu.
Inputting a gametime limit here will show anything at and above the number you enter. For example if you enter in 100 it will show you any servers that are set to 100 hours and above.
Moved item spawns around throughout the map.
Added a Survivors Map to the main menu. To the left of the last level played is a clickable link that when clicked will pull up a map and displays your characters location.
All backpacks have had their weight increased:
Med.back: 15->30
Small:10->20
Large: 20->40
Military: 37->60
Teddy: 10->20
Alice: 30->45
Gamespot backpack: 15->30
Santa: 30->45

Bug Fixes

Fixed few server exploits stacking not stackable items in backpack.
Fixed few server exploits reloading/unloading items in backpack
Fixed a bug that was not equipping items when picked up and an item slot was empty.
Fixed a bug with the nailgun ADS that took up the whole screen while ADS.
Fixed a bug that allowed players to spam prone/crouch to quickly log out of the game (battlelog).
Fixed a collision bug in the police stations.
Fixed a crash inside the med center in Clearview.
