Optimisation des animations pour les MDPS

Message #1 le 2014-4-30 @ 18:28:57 par Xarek:

Plusieurs d'entre-vous le savent déjà mais je vais tout de même essayer d'expliquer le plus simplement possible comment fonctionne la méthode du triple skill burst.

Comme vous le savez, chaque habileté génère une animation dans le jeu et tant que cette animation n'est pas complétée, vous ne pouvez pas faire une autre action de la même catégorie. Cela pose problème dans l'optimisation des rotations car les animations ne sont pas toutes de la même longueur.

Avant de vous expliquer comment optimiser votre DPS en melee, je vais juste récapituler les types d'habiletés :

NIVEAU 1

- Light attacks
- Heavy attacks

NIVEAU 2

- Habiletés de stamina
- Habiletés de magicka

NIVEAU 3

- Block
- Bash (interrupt)

OPTIMISATION

En partant du concept que chaque niveau supérieur annule l'animation du niveau inférieur, il est facile de comprendre qu'on peut "empiler" jusqu'à 3 attaques en 1 en respectant le bon ordre :

1) light attack
2) skill instant de magicka ou stam
3) bash (interrupt).

C'est autant utile en PvE qu'en PvP car la différence de DPS est quand même significative et ce, même si vous n'avez pas une spec orientée bash.
