Chang? la langue du jeu

Message #1 le 2008-11-21 @ 12:20:11 par Meitral:
Saleut all !

Mon frere est vraiment pourri en francais. Il avais essayer sur les server US, mais il comprenais pas les quest, test, etc..etc.. Alors il avait embarquer sur les serveur EU, avec le jeu en francais.

Tant qu'a moe j'aimerais ca qu,il embarque sur WAR, le potentiel du jeu est 20000 fois meilleur que WoW. Le probleme etant toujours la langue ou la location des serveur, ca fait un ptit bout que je fouille pour trouver une solution. Et puis... Bang ! j'ai trouver a matin !

il y aurais une commande /language. Autremendit /language 1 serais anglais, /language 2 serais francais pis /language 3 ben...j'ai pas porter attention pcq j'n'ai pas de besoin.

J'ai pas tester encore si ca marche ou pas, jsuis a job en ce moment.

Yaurais aussi /language french ou /language english qui pourrais marcher aussi...

Et, l'autre alternative, c'Est d'allez jouer dans le fichier usersettings.xml du repertoire USER, en bas du fichier il devrais y avoir une ligne de prametre pour la langue, changer le chiffre 1 pour 2 si vous le voulez en francais.

Pour tout changement, vous devez redemarer le jeu et probablement downloader un patch avec le contenu du jeu en francais ou autres langues, d'apres ce que j'ai lu c'Est proche de 200 megs.

Comme je disais tantot, j'ai pas encore tester voir si ca marche ou pas, les info etais sur des site francais francais de la france. Peut-etre que ca marche juste sur les jeux version EU. Anyway, si ca peut aider du monde tant mieux !

Si quelqu'un a le temps de tester ca avant moi ce soir, ca serais pas pir d'avoir du feedback, merci !

Salut bye !
