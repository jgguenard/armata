Message #8 le 2012-9-30 @ 13:20:17 par Alpro:
Une autre mise à jour, le lien est dans le premier post:

Demo 4.4
- Basic shot/throw ability: increased cost from 1 mp to 2 mp
- Hunter, warrior and assassin armors: ranged mp cost -0.5 rather than -1
- Warrior's scar: +10% critical hit and block chances rather than +5%
- Assassin's mask: no more +15% block chances with high mobility weapons
- Assassin's mask: bleed chances while using sharp weapons 15% rather than 25%
- Spells base critical hit chances: increased from 10% to 20%
- Dart spell: 50% chances to have a side effect rather than 35%
- Gamepad: x/y axis thresholds default set to 6 rather than 3
- Bug fix: items in backpack no longer get bonuses upon entering a shop
- Bug fix: music files extension now work properly in uppercase letters (.MID)
- Bug fix: all orbs should now be placed properly in the dungeon (hopefully)
- Bug fix: mp cost display for crossbows now appear properly (hopefully)

Demo 4.5
- Dart spell: side effect chances back to 35% rather than 50%
- Lance spell: side effect chances set to 20% when not criting
- Gamepad: "Cancel" now also displays "(press button to assign)"
- Bug fix: mp cost display for crossbow abilities (now includes fast reload)

Demp 4.6
- Specter boss: casting speed 24 rather than 25 (that means faster)
- Specter boss: increased summon bats delay
- Specter boss: modified chasing behavior

Votre partie sauvegardée (devilcrag.sav) ne sera probablement pas compatible avec cette nouvelle version, désolé.
