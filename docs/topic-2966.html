Message #1 le 2012-11-2 @ 12:04:10 par drbeefz:
Bonjour,

Je fais ce poste pour vous avertir et aider dans les prochaines semaines.
Beaucoup de vous démontre de l'intérêt pour les RBGs, et avec raison.

Ce que vous devez savoir ?
Seulement certain ''spec'', et d'autres plusieurs, sont utiles en RBGs.
Donc je vous lance l'appelle de vous rendre conforme à ces exigences pour augmenter vos chances et succès en RBGs. Si vous ne souhaitiez pas le faire, libre à vous, mais veuiller tout de même prendre en considération les avantages aux groupes pour le succès, et à ce moment, faite le choix entre plus de succès et plaisir de jouer.

DeathKnight : Frost Frost Frost, simplement, un spec Blood peut dépanner, mais pas nécessaire

Druid : Tous spec sont utiles, une de seule classe, par contre, ils seront utiliser majoritairement en Resto. Travailler sur un deuxième spec Balance ou principalement Balance

Hunter : Principalement BM était le choix, reste à voir, je crois que Marksmen va revenir le spec à aller pour. Survival dans le cas que vous aller servir principalement de classe défensive aux nodes maps

Mage: Typique, Frost ou Fire, lorsque seulement un mage, il se doit d'être Frost majoritairement

Monk : Mistweaver principalement, vous devez avoir une instance de ce spec, vous remplacer le Paladin holy à ce moment. Une instance de Windwalker peut-être utile dans les maps Flag Carrier

Paladin : Principalement Holy en tout temps, un spec Ret peut-être utile lors du manque de Frost DK et/ou certaines maps pour des poursuites d'orbes, flag, etc., mais Holy principalement

Priest : En terme de choîx, aller Shadow. Un Disc ou Holy offspec peut dépanner, mais Shadow pour l'instant

Rogue : Assassination sur les maps de nodes, Sub sur les maps des orbes et Flag Carrier. Vous devez avoir les deux.

Shaman : Élémental ou Restoration, il vous faut les deux. Une principale est ok, mais un hybride est impératif dans les plus hauts classements

Warrior : Vous allez être utilisés majoritairement en ''Flag Carrier'', vous devez avoir un spec Protection, et investir dans cet équipement itemisation (gems, enchants, shield et one hand ainsi que trinkets)

Warlock : Destruction, vous devez travailler sur ce spec. Un spec affliction pourraient toujours venir utile dans le scaling de gear et comme rôle défensif, mais pour nos compositions, Destruction sans fautes

Merci et bon succès,
Drbeefz
