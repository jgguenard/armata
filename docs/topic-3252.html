Tasos patch notes 7 fevrier

Message #1 le 2013-2-7 @ 12:48:59 par Braaw:
Quote: These are the patch notes for today's patch. If you haven't yet, please check yesterday's update on the reasoning behind some of the patch note items.
Fixed a bug where mobs became unresponsive.
Added the ability to combine one-handed weapons and shields in the quick item bar: The player can now activate these items simultaneously from the bar.
Treasure maps (Click on a treasure map to start a treasure hunt! Your minimap will start flashing: Green if you are moving closer to the treasure and red if you are moving away from the treasure. Use a shovel to dig the treasure out once you´ve located it.

[size=large]Game Balancing:[/size]

We reduced base vital stats penalties for roles:
Warriors gain 15 Stamina, 30 Mana (Total of 45)
Skirmishers gain 15 Health, 30 Mana (Total of 45)
Elementalists gain 15 Stamina, 30 Health (Total of 45)

Increased Bow reticle rest time by 100 milliseconds for Warrior
Increased Bow reticle rest time by 100 milliseconds for Elementalist
Reduced Staff reticle rest time by 100 milliseconds for Elementalist
Increased Bow targeting randomness by 1 radian (run) for Warrior
Increased Bow targeting randomness by 2 radians (sprint) for Warrior

Doubled Staff Bolt Mana Cost
Reduced Staff Bolt area of effect radius
Increased Staff Bolt Missile Speed by 12.5%

Added Casting Time Bonus (max 10%) to Attunement to Fire.
Reduced Duration of Attunement to Fire by 50%
Reduced Cooldown of Attunement to Fire by 50%

Reduced Warrior Armor Magic Protections (by 8.3% on average, lightning 6.6% on average)
Reduced Skirmisher Armor Magic Protections (by 10.1% on average)
Increased Elementalist Armor Piercing Protections (by 85.4% on average)

Players will gain from 25 to 35 of each Health, Mana, Stamina when killing a player (not ganking)
While resting a player will suffer a 50% protection penalty

[size=large]Agriculture:[/size]

Increased Potato drop rate when digging by 150% and quantity by 33%
Increased Carrot drop rate when digging by 25% and quantity by 33%
Increased Onion drop rate when digging by 50% and quantity by 33%
Increased Fossil drop rate when digging by 100% and quantity by 100%
Reduced Minimum distance for chance of successful digging to 35m (30% reduction)
Added new Potato Plants to Agon as a random spawning object.

[size=large]Added sounds:[/size]

Paperdoll operations
Skill operations
drag/drop/delete operations
Gold
Bindstone/House Recall

[size=large]Various Fixes and Additions[/size]

Removed Wisdom prerequisite from Elementalist Robes
Please Note: In general any dual prerequisite is either or e.g. either STR or DEX for 1h weapons. This is to clarify the misconception that the players need both.

Fixed a bug in the Radial Options Menu where the text sometimes became overwritten.
Added double click on clan holding markers on the political map, when a player double-clicks on a Clan Holding marker, the Clan Holding overview window opens.
Fixed so that only the progress bar is visible when in GUI mode
Added max character limit of 20 in personal tag editor
Enlarged the input area in the personal tag editor
Fixed key bindings problems in the personal tag editor
Added bindstone and house recall buttons to minimap
Changed icons for bindstone and house recall buttons
Changed the icons for safe area protection
Added progress indicator for safe area protection activation and deactivation.
Fixed so that non-political map related markers do not appear while in political map mode.
Fixed so that tracked feats are now persistent.
Added SFX to revive
Fixed a bug where sometimes the block animation wouldn´t play.
Fixed so that a selected skill will not revert to default, when pressing the attack button(LMB) while another skill is running
Fixed so that if you click and hold the attack button while a spell is in the progress of being cast, you will now be able to cast the next spell by holding down the button: i.e. the attack will not be blocked.
Fixed a bug that caused the wrong calculation of backpack weight in the GUI
Fixed an exploit where the player would dismount while backing into the surroundings, causing him to spawn inside geometry.
Removed monster collision from dead monsters
Fixed an issue where skills with long cooldown did not update fast enough in the GUI
Changed vegetation options. (Range and density)
No skill ups on mounts (except heal mount skill)
Mount protection in protected area bug fixes.
You can now turn around while you are changing your role.
Added visual effect for Player Summon.
Summoner will no longer lose the portal shard if target player is in a dungeon.
Fixed the mount turning glitch.
Show system messages for unlocked,completed feats.
Player gets continuous audio and visual notifications while getting environmental damage.
Players can no longer detach from a moving ship.
A player can no longer be summoned while on a mount or attached to a ship.
All boats will get a 10 minute ownership timer after their initial spawning or after a player detaches from them provided the boat is located in a protected area.
Players will no longer be able to fire a ship cannon while they are in a protected area.
Some changes in parties in order to fix the lag.
"Summon friend" cooldown does not reset anymore after player re-logs in game.
Fixed input binding of window hot keys.
Fixed video options bug where certain settings were not saved correctly.
Increased vegetation density.
Minor engine optimizations.
Updated description for Static Field
Updated description for Lightning Bolt
Updated name and description for Herb Spawn Standard
Updated name and description for Mining Spawn Rock
Updated name and description for Mining Spawn Metal
Updated name and description for Logging Spawn Tree
Updated accessories for Alfari.
Updated hairstyles for Mercians
Updated hairstyles for Mahirim
Updated hairstyles for Tovarr
Updated Tovarr beard
Updated Mercian Female fishing animation
Fixed shovel placement on all races
Small visual enhancement to the crosshair

Thank you for reading
