La Charte de l'Alliance Québec

Message #1 le 2009-10-3 @ 15:03:44 par Scoute:
Tout les points mentionnés ci-dessous peuvent changés sans préavis des membres , Donc il est important de les lires chaques semaines.

1- Chaques membres fesant partit de cette alliance est dans l'obligations de s'inscrire sur le forum d'Armata Tenebrae. ( un délais de 1-2 jour est accordé pour créé l'account forum par l'admisnistrateur du site internet.)

2- AUCUN INVITE INTER-GUILD SANS CONSENTEMENT DES DEUX PARTIS ( Seul les guildes leaders pourront autoriser les changement de guilde avec l'accord de tout les officiers / Leaders de l'alliance )

3- MINIMUM D'UNE PERSONNE RESPONSABLE PAR GUILDE ( Cette personne devra être très active pour éviter d'attendre 2 semaine pour une réponse a un probleme inter-guilde.)

4- Spam, bitchage, impolitesse, language vulgaire, racisme = retrait de permision de parler dans le chat de l'alliance ( Officier/GM de guild concerné informer et conséquence exigé. Trois avertissement et ensuite des mesures drastiques seront exigées.)

5- UN ROSTER ( Lien url ) sera exigé par le / les responsable de chaques guildes. ( Ceci a pour but de créé une liste des personnages levels 40 selon leur équipement pour faciliter les groupes inter-guilde. )

6- TOUT ÉVENEMENT ORVR EXIGE LES PARTICIPANTS A SE CONNECTER SUR VENTRILO. ( les informations du ventrilo seront précisé sur l'alliance chat. )

7- Toute modification de cette charte doit être approuvé par Conehead avec l'accord des autres guilde leader impliqués.
