Message #1 le 2011-11-13 @ 11:40:00 par Primex:
Bonjour j'aimerais vous présenter un UI très pratique qui me fait penser au bon vieux Power Aura de WoW.

En gros c'est un mod qui réagit a certain évenement que vous lui spécifier. La facon dont le mod réagit c'est vous qui le décidez. Que se soit des images importer par vous, des textes ou des timers.

-Ton pet vien de tomber a 70% hp, un ptit point d'exlamation aparait

-Quand j'applique un DoT sur un mob une icone aparait avec le timer restant sur le DoT (même chose pour les debuff ou les HoT sur un target)

-Un boss cast une abilité (Pratique pour les interupt, dispel et purge)

-Une abilité tombe en cooldown (avec le timer) ou revient de cooldown (l'icone pop et dure un certain temps)

-Il manque un buff sur le raid, un message aparait ''missing Fervor on raid'' (Pratique pour les buffeux qui veulent pas se faire taper sur les doigt)
Se petit mod va simplifier votre UI. Si vous êtes comme moi, je cherch un UI le moins achalander possible avec le plus d'information dans le même endroit pour faciliter le gaming. Se mod devient alors un must avec le UI présent de Rift.

Le mod : http://www.riftui.com/downloads/info103-WarnMe.html
