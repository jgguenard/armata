Warrior Tanking spec Analyse

Message #1 le 2011-11-20 @ 11:11:42 par Primex:
Mixte d'analyse de plusieur build qui peuvent/pouraient plaire au warrior d'aramata. Les top build de Mitigation se retrouve au bas de la liste. Le spreadsheat utiliser pour faire les calculs se trouve ici :

https://docs.google.com/spreadsheet/ccc?key=0AjsoNPZHw4INdHFQdi1rbzVjQTlSQlBsRF9xYlJ0eGc&hl=en_US



Build #1: 51 VK/15 PL - Pure Void Knight

Physical Mitigation: 74.4%; Magic Mitigation: 38.2%; Block 67%
Comments: Vien avec de 'tools' pour aider dans les plus petit groupe. Tres facile de garder les pact avec accord of emptiness et chance de rafraichir le void summon (maintenant un taunt) on hit. Selon moi il y'a 7 point de perdu dans se build.

Build #2: 51 WL/15PL - Pure Warlord

Physical Mitigation: 71%; Magic Mitigation: 39.2%; Block 68%
Comments: Un build a revoir. Emmene un cooldown tres bien avec No permission to die. Le skill call to regroup es tres utile aussi. Malheureusement je ne crois pas que passer 32 point dans Warlord vallent la peine.

Build #3: 51 PL/15 WL - Pure Paladin

Physical Mitigation: 70%; Magic Mitigation: 34%; Block 73%
Comments: Le pire build de tank coté mitigation. Utiliser sur un fight physical seulement en offtank se build peu shiner, that it. Il emmene 3 cooldown incroyable quand même. Preservation es un buff qui dure jusqua la mort et qui reduce le dmg taken de 10%, a mettre sur le MT. Impassable Guard bloque TOUTE les ataques physique et reduit le dmg magical de 25% pour 10 sec, 1 minute CD. Le best 44 point CD imo. Finalement Life rapture reduit le dmg de tout le monde de 10 personne dans le raid de 10% et les heal de 100% de la vie du paladin durant 6 sec... wow oO. Dommage que le build soit si crappy coté mitigation passive, sinon sa aurait été un bombe.

Build #4: 38PL/28WL/0VK - Offtank Buiild

Physical Mitigation: 74.3%; Magical Mitigation 38.1%; Block 75%
Le spec de offtank par exelence. Emmene 10% de dmg reduction sur le Main tank et il va chercher Aid command qui increase le healing done de 20% pour 26 seconde. Une bonne alternative es de dumper le 10% de dmg reduction dans paly pour aller chercher le 31 point de warlord qui increase le maximum ET CURRENT health de tout le raid de 25% ainsi que leur resistance par 30 pour 26 seconde.

Build #5: 26 VK/24 RV/16 WL - Shield Hybrid Modifier

Physical Mitigation: 80%; Magic Mitigation: 52%; Block 51%
Comments: L'ancien build de shield tank. Un bon build solide all around, de tres bon utility. Aucun 1 minute cooldown, se qui le rend un peu moin désirable in raid. Un gros trade off en perdant bcp de bloc pour gagner de la passive mitigation.

Build #6: 21 RB/16WL/29VK - Blink Tank

Physical Mitigation: 77%; Magic Mitigation: 43%; Block 51%
Comments: Un build de dmg reducion quand même asser basse. L'avantage es simple, beaucoup de tools, un bon dps si besoin et énormément de mobilité/taunt. Nécésite de la configuration et beaucoup d'experience. Personellement je ne l'utilise que pour les 5 man rush.

Build #7: 26 RB/16 WL/24VK - AoE off tank Build

Physical Mitigation: 75% Magic Mitigation: 51% Block 51%
Comments: Une variante du build tank faite experessement pour l'aoe tanking. Vous gagner un taunt/pull toute les 6 seconde grace a Planar blade. Vous disposer de pratiquement infinie energie (thx planar blade again). Vous pouvez mettre Freeze armor pour les autre warrior (20% more dmg sur leurs finisher). Vous avez aussi l'ancienne version de spoted order avec Surge et Pact conversion.

Build #8: 44 VK/22 WL/0 PL - The VoidLord

Physical Mitigation: 76%; Magic Mitigation: 45%; Block 62%
Comments: Un build exelent surtout pour tanker des 5 man/10 man. Beaucoup d'utilité et les meilleur cooldown average coté warrior tanking. Le build que j'utilisait quand 1.6 a été lauch.

Build #9: 44 PLd/22 WL/0 PL - PallyLord

Physical Mitigation: 74%; Magic Mitigation: 41%; Block 73%
Comments: Un exelent build dans un fight entierement physique. Emmene le meilleur des 44 point talent avec impassable guard et donne aussi le 10% dmg reduction sur un autre tank. Un des plus haut total de block rate avec en plus un 20% d'absorb sur le block!

Build #10: 44 RV/22 WL/0 PL - The ReaverLord

Physical Mitigation: 83%; Magic Mitigation: 43%; Block 58% - Avec Potm
Comments: Le spec de base deep reaver. Infestation es vraiment bien pour 1 minute coolodwn. Vous obtenez enraged essence pour le grp aussi!

Build #11: 44 VK/22 RV/0 PL - The Death Knight

Physical Mitigation: 83%; Magic Mitigation: 48%; Block 58% - avec POTM.
Comments: Un build avec beaucoup de cooldown defensif et d'utilité. Avec ravenous defense on peu réussir a obtenir un taux de bloque correcte. Tres bonne mitigation de base.

Build #12: 44 PL/22 RV/0 PL - The VoidLord

Physical Mitigation: 82%; Magic Mitigation: 44%; Block 68% - avec POTM
Comments: Un exelent build de tank, et encore plus de offtank. Comme tout les build de 44 paly il emmen 2 incroyable cooldown. Beaucoup mieux en reduction magique que les autre 44 build de paly mais perd intercept et cutting distraction.

Build #13: 44 RV/16WL/6PL - Cooldown Reaver

Physical Mitigation: 79.3%; Magical Mitigation 54.1%; Block 70%
Comments: Le build que j'tuilise présentement. A quelque point des top mitigation build mais avec quelque modification que j'utilise encore même dans les raids 20 man.

Build #14: 38RV/28VK/0PL - Max Mitigation

Physical Mitigation 80%. Magic Mitigation: 55.1%; Block 58%
Le build non-block mitigation in game. De très bon cooldown qui encore une fois viennent du Void knight. Aucun 44 sec cooldown malheureusement se qui peu être utile sur certain fight. Des que vous avez le 4 set bonus de HK warlord il serait mieux de passer a la variante du warlord suivant

Build #15: 38RV/16WL/12PL - Top Average Mitigation

Physical Mitigation 79.9%; Magic Mitigation 54.1%; Block 70%
Le build de mitigation passif ultime pour les warrior top gear. Des que vous avez le 4 set HK warlord crystal vous switcher pour se build et beneficier de la meilleur protection passive in game.



Conclusion : Il es difficile de se tromper en fesant un build depuis la patch 1.6. Vous pouvez faire pratiquement n'importe quoi en restant dans les soul tree de tank et arriver pratiquement a la même mitigation que pre 1.6 grace au gift talent disponible dans chaque tree de tank.

Overall pour un max mitigation build vous voulez inclure 38 reaver maintenant. Tout les tree on un 1 minute cooldown qui reduce le dmg taken de 35% (different pour paly) pour 10 seconde.

Le 31 point warlord a légerement perdu sa place en tant que top offtank build, contre le 44 point paly qui emmene d'exelent cooldown pour le raid.

L'ancien build de 51 point void knight n'a plus réelement sa place parmis les tops. Il reste certe un très bon build pour se faire du fun, mais n'apporte pas la top mitigation comme avant.
